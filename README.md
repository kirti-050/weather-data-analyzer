# Weather Data Analyzer

A Python project that fetches real-time weather data using the wttr.in API and analyzes it using Pandas.

## Features

- Fetches live weather data
- Extracts temperature, humidity, pressure, wind speed, and weather description
- Stores data in a Pandas DataFrame
- Exports weather data to CSV
- Generates simple weather insights

## Technologies Used

- Python
- Requests
- Pandas
- CSV
